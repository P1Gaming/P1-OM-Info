Detailed Description

Inherit from this abstract base class to create controller components that respond to input system events.

When enabled, a Controller component will begin listening for input events. When disabled, it will stop listening for such events. Concrete derived controllers must define which input events to listen for.

Attempts to register with the input system as a listener for player control events.

Gracefully fails if unable to find the PlayerInput component or required input actions, in which case the character simply does not move in response to any input.

Implemented in Player.View.ViewController, Player.View.ViewChange, Player.Motion.Movement, and Player.Motion.Jumping.
